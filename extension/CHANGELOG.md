# Changelog

## 1.1.1

- Added a prompt for the host to choose the shared document name when creating a room.
- Opened the collaboration scratch document using the chosen filename instead of a generic untitled tab.

## 1.1.0

- Switched the default server URL to the hosted backend.
- Added the getting started panel for first-run onboarding.
- Added `Copy Student Invite Token` so hosts can re-share invites after reloads.
- Improved connection, invite, protocol, and room availability error handling.
- Added marketplace-ready PNG icon and packaging metadata.
- Added protocol handshake support for hosted deployments.
