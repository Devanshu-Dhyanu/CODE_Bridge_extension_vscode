# Support

## Need help?

1. Open `CollabCode: Open Getting Started` for the default host/join flow.
2. Check the hosted backend health endpoint: `https://code-collab-5qo3.onrender.com/health`
3. If the hosted service is healthy but you still cannot connect, open an issue at:
   https://github.com/Devanshu-Dhyanu/CODE_Bridge_extension_vscode/issues

## What to include in a bug report

- VS Code version
- CollabCode version
- Whether you are using the hosted backend or a self-hosted server
- The exact command you ran (`Create Room`, `Join Room`, etc.)
- The error message shown in VS Code
